## Noah's Resonance
This is a VR music game. Players will take on the role of a "Bard" who commands the laws of sound. In a perilous space environment, you will decode alien languages and communicate with unknown lifeforms through rhythm-based gameplay, ultimately saving Earth's civilization from the brink of collapse.

## File Description
* *Arduino/*：Hardware control code for storing ESP32.
* *Unity/*：Stores the script codes for Unity games/interaction ends.
* *teamCobalt/*：Stores all the assets of this game. [teamCobalt Drive](https://drive.google.com/drive/folders/1yOq3qJV-no6fXi57-V8RxlNnkDmcb35e?usp=drive_link)

## 运行环境
* Arduino IDE (ESP32 核心库)
* Unity 202x.x
